# Unisender wordpress subscription plugin

### Что дает интеграция WordPress с Unisender
Плагин Unisender для WordPress обеспечивает простое и удобное решение для создания формы подписки на вашем сайте. С его помощью вы можете легко собирать контактную информацию от посетителей вашего сайта и автоматически добавлять их в ваш список рассылки в Unisender.

### Основная функция
Основной функцией плагина является встроенная форма подписки, которая позволяет посетителям вашего сайта легко подписываться на рассылку. Форма подписки полностью настраиваема, чтобы соответствовать дизайну и стилю вашего сайта.

### Что нужно для интеграции
Для использования плагина вам понадобятся аккаунты в WordPress и Unisender, а также API-ключ от Unisender.

### Где взять API-ключ Unisender
Для получения API-ключа Unisender, пожалуйста, посетите официальный сайт Unisender и следуйте инструкциям для генерации ключа в вашем личном кабинете.

[Где взять API-ключ Unisender](https://www.unisender.com/ru/support/api/common/api-key/)

### Как подключить Unisender к WordPress
- [Скачайте архив плагина](https://gitflic.ru/project/unisender/unisender-wordpress-plugin/file)
- В административной панели WordPress перейдите в раздел «Плагины» — «Добавить новый» и нажмите на кнопку «Загрузить плагин».
- В появившейся форме выберите ZIP-файл с плагином на вашем компьютере и нажмите на кнопку «Установить».
- После успешной установки активируйте плагин.

Если у вас возникли проблемы на любом этапе установки или настройки, вы можете обратиться за помощью в техническую поддержку Unisender по ссылке: [Техническая поддержка Unisender](https://www.unisender.com/ru/technical-support/)

С плагином Unisender для WordPress ваша email-маркетинговая кампания станет еще проще и эффективнее. Установите плагин сегодня и начните привлекать больше подписчиков на свой сайт!
